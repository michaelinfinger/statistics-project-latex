# statistics-project-latex
This repository is the final project for my Statistics course. I chose to format it in $\LaTeX$ to learn more about the programming language in order to facilitate upcoming projects in the language. This project is an analysis of a survey given to two high schools in the Walton County area (SWHS and MIC), and it attempts to pick apart associations in the data.

## Plan

1. I will create the barebones of the project, including hypotheses, general hypotheses, and formatting. I will also formulate and refine my survey.

2. I will distribute my survey via several methods, such as QR codes, peers, and rewards.

3. I will import the data in a CSV to make it readable by pgfplots and other latex packages.

4. I will then import the data into the document via a table and showcase it in its entirety.

5. I will analyze the data and find associations

6. I will make my conclusions, reject hypotheses, draw connections, and refine my project.
